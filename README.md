# node.js 란
자바스크립트는 웹브라우저에 내장된 자바스크립트 엔진으로 실행됩니다. 그래서 자바스크립트를 
실행하는 웹브라우저를 자바스크립트의 구동 환경이라는 뜻에서 '자바스크립트 런타임' 이라고도 표현합니다.
 한 마디로  Node.js는 또 하나의 자바스크립트 런타임입니다. 따라서 node.js는 웹 브라우저 외에서도
자바스크립트를 실행시켜줍니다.
